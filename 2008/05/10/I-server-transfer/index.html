サーバ移転しました

May 10th, 2008

やっと，サーバを移転しました。

でも，実家にあったサーバをこっちに持ってきてしまいました。

まぁーサーバ移転（筐体の入れ変え）なんて，私にすれば単に気分転換であり，浪費でしかありません。

＝＝＝＝＝＝

私の浪費の結果を以下に書いてゆきます。

（腹減ったなー）

新筐体：DELL SC440 (￥２４，０００-) + Xenの完全仮想化を実現する為にE6600中古(￥１６，２６１-)＋メモリ入れ替え（￥３，９８０）＝４４，２４１円

まったく無駄遣いです。

ちなみに，コンソール用にLCDを買いました（￥７，６００-）

（うはっ）

OS（オペレーティングシステム）：CentOS 5.1

メモリ：2GB

HDD(ハードディスクドライブ):８０GB(RAIDなし)

[Xenを導入]

Domain-0：CentOS 5.1

Domain-U：Windows xp (Mem:512MB,HDD:20GB)

[管理方法]

CentOS 5.1 : FreeNX(Xfce4)

Windows xp : リモートデスクトップ

[サービス]

WWW（lighttpd）

メールサーバ（構築中）

あとなんか

＝＝＝＝＝＝

今までの環境をばっさり切り捨てた為に技術資料というか，作業手順書を作り直しています。

お腹が減ってどうしようもないので，途中ですが，続きはまた今度。
