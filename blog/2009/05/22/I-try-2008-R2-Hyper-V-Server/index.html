Hyper-V Server 2008 R2を試してみる

May 22nd, 2009

　先日かいしゃの面談がありました。 　もっとしっかりアピールするようにだそうです。 　まったくそのとおりだとおもいましたｗ 　＝＝＝＝＝＝＝＝ 　広く深く（リナックスもウインドウズも）知識を身につけては？ということでした。 　私はネットワークエンジニア志望ですよ。そんなアバウトなーなんて思いながら、無償で利用できる「Windows Server 2008」の機能限定版である「Hyper-V Server 2008 R2 RC」をダウソして眠っていたSC440にインスコしてみました。 　＝＝＝＝＝＝＝＝ 　インストールは簡単そのもの。 　CDを入れてブートして、言語とキーボードを選択してOKをポチポチで入ります。 　パスワードを設定して、ログインすると・・・コマンドプロンプトが２つしか動いていません。 　そう、Server Core だけでシェル（Explorer）は入っていませんｗ 　しかし臆することはないんです！！ 　ほとんどの操作がmmcを使ってリモート管理できます。 　Hyper-Vに関しては、Hyper-Vマネージャーを使えば問題なしです。 　＝＝＝＝＝＝＝ 　Windows 7 RCが評価版で手に入るので、mmcやらHyper-Vマネージャーは直ぐにセットアップできて、使えるようになりました。 　Windows Server 2008といってもWindowsです。 　IEはインストールできませんが、Firefoxならインストールできます♪ 　Firefoxも64ビット対応を入れないと日本語変換が使えませんが、入れれば普通に使えますｗ 　＝＝＝＝＝＝＝ 　Windows 7はVMware Playerで動かします。 　こいつほんとに良くできています♪ 　＝＝＝＝＝＝＝ 　今回わかったこと 　Server Coreだけでも、ネットワーク周りとディスク周りはもちろん「サーバ」用なので、Server2008で動くソフトウェアを入れてやれば、ただで安定した○○サーバが作れるようです。（が使ってはいけない。） 　とても興味が湧きましたよ！！ 　＝＝＝＝＝＝＝＝ 　こうなると今の３台の仮想環境をXenからHyper-Vに移し変える日も遠くなさそうです。 　参考にしたサイト 　・PaulDe&squot;s Blog : http://blogs.technet.com/pdespe/archive/2008/11/20/workgroup-hyper-v-server.aspx 　・エンジニアの魂：http://d.hatena.ne.jp/s0u/20081004/1223133499 　・Sugishita&squot;s Blog : http://blogs.dandunite.com/sugishita/post/Hyper-V-Manager-on-Windows-7.aspx 　・furasuke21の日記 : http://d.hatena.ne.jp/furasuke21/20090513/p1
