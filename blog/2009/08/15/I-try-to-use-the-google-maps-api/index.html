Google Maps Api を使ってみる

Aug 15th, 2009

さいきん夏バテの林です。

９月に実家に帰るので、ちょっと前に作って数回しか使えなかった「携帯電話からブログの記事を書く with GPSデータを埋め込んだ写真とgoogle mapsを組み合わせる」をもう一回作ってみることにしました。

日々成長を求められるわけですが、前回とは違って複数マーカーが表示された時のセンターとズームをjavascriptで計算して自動で行うようにしました。

PHPでちょちょいと計算して静的に埋め込めばいいんですが、ちょっとjavascriptを駆使してみましたw

ここでサンプルコードとかサンプルのマップを乗っけることができればいいんですが、腹が減ってダメです。
