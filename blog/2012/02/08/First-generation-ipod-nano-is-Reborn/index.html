初代 Ipod Nano がリボーン

Feb 8th, 2012

当時（2006 年）に背面が鏡面加工された ipod nano （初代）の 2GB モデルを 20,000 円くらい出して買ったのを思い出します。 カラーでしたが小さい画面で電池の持ちがわるく、音も弟がもっていた Panasonic の SD プレイヤーよりも劣悪でこれが Apple 品質か！！と SONY の WALKMAN を布教してまわってましたのも思い出します。 まったく使ってなかったので、状態もキレイで捨てるにももったいなく、電池の持ちが悪いので貰い手もなくでしたが、なぜかなくならず目の届くところにいつもあったのが今でも不思議でした。 が、ある日こんなのを見つけました。 アップル、バッテリー過熱で初代iPod nanoの交換プログラムを発表 : http://www.gizmodo.jp/2011/11/ipod_nano_19.html 初代iPod nanoを交換してもらったら第6世代nanoが送られてきた！ : http://www.gizmodo.jp/2011/12/apples-recalled-first-gen-ipod-nanos-are-now-being-replaced-with-new-models.html おおおおおーと騒いでいると、まわりでも事例がちらほらと出てきて、私ものってみました、このビックウェーブに うん、届きました。 第6世代はタッチパネルでラジオも付いてるんだーなんて考えながら充電してます。 しかながら事前情報にあったようにサポートは受け取った途端にサポート終了となりましたー おまけ こんな使い方もできるそうな iPod nanoを腕時計として利用する技 : http://ascii.jp/elem/000/000/641/641344/
