Aspire 1410 に Linux Mint 12 を入れた

Feb 20th, 2012

　メインで使ってる ネットブックを Ubuntu 11.10 から Linux Mint 12 Gnome に入替えました。 　動機は DistroWatch でラインキング1位になっていたから… 　DistroWatch.com : http://distrowatch.com/ 　Ubuntu は重たいらしく、すごい差がついていましたよｗ 　この時間になるとヤル気がでるので、簡単にやったことまとめるよ ### インストール 　※ CD (or DVD) に焼くんじゃなくて USB メモリで作業をすすめる 　Download Linux Mint 12 Lisa : http://www.linuxmint.com/download.php 　1. メモリ 4 GB あるので DVD 64-bit をダウンロードする 　12 になって”ハイブリットISO”とやらになって dd してもいけるというのでやってみるも、not active partition ? とかでるのでいつもの 　UNetbootin – Homepage and Downloads : http://unetbootin.sourceforge.net/ 　2. Linux 版をダウンロードする 　※ USB メモリを FAT32 でフォーマットしておけば認識されるはず 　Installation & Screenshots : http://unetbootin.sourceforge.net/#install 　3. Distribution で 「Linux Mint」 「10_Live_x64」 で落としてきた ISO を指定して書きこむ 　Official documentation : http://www.linuxmint.com/documentation.php 　4. USB メモリでブートしたら、このへんを見てインストールすすめる 　※ ユーザフォルダが日本語になるといちいちイライラするので English でインストール 　5. ベースは Ubuntu 11.10 なので、インストーラの雰囲気もおんなじでなんなくログイン 　Ubuntuの日本語環境 : http://www.ubuntulinux.jp/products/JA-Localized 　6. 「方法２・Japanese Teamのパッケージレポジトリを追加する」 以下をコピペですすめる 　Linux Mint 12 “Lisa” と Google 日本語入力 : http://wellvil.com/blog/archives/2011/1127194836.html 　Linux Mint 12 を使いやすく設定する : http://shiroichi.sakura.ne.jp/2011/12/20111224mintsetting/ 　7. メニューとかの日本語かとか日本語入力とかを整える まーこんなもんで、さあ寝るか ### 番外編 最近 こんな使ってます YouTube Streamの目覚しい進化｜3G通信対応、バックグランド再生、Twitter投稿 : http://d.hatena.ne.jp/RyoAnna/20100902/1283445248
