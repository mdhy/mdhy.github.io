W2k8 + Utorrent ってどうよ

Sep 18th, 2009

こんばんは。

週末の旅行に備えて洗濯とか冷蔵庫の整理とかをしている林です。

世の中にはBitTorrent（ビットトレント）という技術があって良しも悪しも世界中で使われています。

私も度々インストールし直して使うことがあるんですが、暇を持て余してまたインストールした話です。

＝＝＝＝＝＝＝

今まで何度も書いてきたようにHyper-V Server 2008 R2を使って3つほどOSを常時稼動させているんですが、リソース（資源）をあまらせています。

（あっ洗濯1回目終わった）

そこで4つ目をインストールしてみようと思い立ち、以前にも使った事のあるuTorrentを使うことにしたのはいいのですがOSは何にしようかと考えるわけです。

見つけましたよ。Windows Server 2008 R2 RCｗ

RC（製品候補版）でも1年は使えるはずなのでインストールを始めてWindows Server 2008 Enterprise Server-Coreを選んで30分くらいでインストールは完了しました。

Windows Server 2008になってから何が困ったちゃんかというとファイアウォールが固めてあって思うように外部と通信ができないことです。

さらにこれがServer-Coreとなるとリモートデスクトップを有効にするのでさえ

cscript c:\windows\syste32\scregedit.wsf /ar 0

上記のようにコマンドプロンプトから操作を行う必要があります。

まぁログインするとコマンドプロンプトがひとつしか出てないし、いろいろと使えないものがたくさんありましたｗ

外部の端末からMMC管理を行えばいいようなんですが、「アクセスが拒否されました」の繰り返しでうまくいかないので断念しました＞＜

＝＝＝＝＝＝

次にutorrentのインストールを行うわけですがコマンドプロンプトからutorrent.exeを実行してやればすんなりインストール完了です。

torrentクライアントの機能として正常に稼動していますが、フォルダダイアログは正常に動作してませんでした。

まぁもうちょっと使ってみないとわからないんですが・・・様子見ですね。

日本の宇宙開発

Sep 6th, 2009

実はロケットとか大好き林です。

９月１１日の午前１時３０分から今後の日本の宇宙事業の発 | 展を占うHTV／H−llBの打ち上げがあります。

JAXA | HTV/H-llB特設サイト（http://www.jaxa.jp/countdown/h2bf1/index_j.html%EF%BC%89

HTVとは、宇宙ステーション補給機（H-ll Transfer Vehicle: HTV）というもので地上から宇宙に補給物資を送り出す無人の輸送機のことです。

アメリカがスペースシャトルの打ち上げを2010年で終了するという発表を受けで世界中（ヨーロッパ、ロシア、韓国、アメリカもスペースシャトルからロケットへ）が宇宙に補給物資を打ち上げるロケットと輸送機の開発に力を入れているようです。

アメリカもスペースシャトルを使った補給物資の打ち上げよりもロケットにカプセル型有人宇宙船か補給船を乗っけた方が費用対効果があるということで部品点数が多く保守に手間のかかるスペースシャトルよりも大気圏脱出や大気圏突入の時に抵抗を受ける面積の少ないカプセル型になるようです。

表面化したスペースシャトル引退延期 – ビジネスオンライン（ http://www.nikkeibp.co.jp/style/biz/feature/matsuura/space/080918_enki/index1.html%EF%BC%89

＝＝＝＝＝

時期が時期だけに今回のHTV/H-llBの打ち上げは失敗できない状況 です。

ヨーロッパ勢も赤道直下に打ち上げ場をもっているので競合ですが、打ち上げが成功すれば、日本の宇宙事業は明るいこと間違いなしでしょう。

そして、三菱重工の株価も爆騰でしょうかｗ

＝＝＝＝＝

あぁ打ち上げが待ち遠しい。

私は日本の宇宙事業を応援しています。
