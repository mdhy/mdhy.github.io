LOVE LOVE BSD

Dec 27th, 2009

さいきんFreeBSDの存在を思い出した林です。

こんばんは。

さて、ここ最近はLinux(リナックス)で仮想化(Virtualization)を実現するのにKVM(Kernel-based Virtual Machine)を使ってみようと思っているんですが、いろいろ調べているとFreeBSD8.0Rがリリースされているようです。

私が初めて出会ったUNIXがFreeBSD4.9Rでした。

私の家に余っていたPC-98にインストールできそうなのがFreeBSDしか無かったからです。

＝＝＝＝＝

なつかしさも覚えながらVirualBoxにFreeBSD8.0R+Xfce4でインストールしたメモです。

付け加えて、FreeNXで接続してつかえるようにしています。

（1）インストールからパッケージインストール

参考：http://sourceforge.jp/magazine/09/12/10/086210

ジャストな記事があったので参考にしました。

インストールはとくにカスタマイズをしなければ、「A」と「Q」と「X」を押していけばおわりますが、インストールするパッケージは「GNOME」ではなく「Xfce4」を使う関係とFreeNXで「SCIM-Anthy」がなぜか？使えないないで「uim+Anthy」を使うようにしています。

インストールしたパッケージは以下です。

・shells/bash4

・x11/xorg

・x11-wm/xfce4

・japanese/ja-font-ipauigotic

・japanese/uim-anthy

・textproc/uim-gtk

・net/freenx

・www/firefox3

ここまでくれば「xfce4-session」を設定して「startx」を実行すれば、なんの苦もなくXウインドウが立ち上がってきます。楽しい

＝＝＝＝＝

けっこう簡単にできてしまったので、FreeNXを使えるようにしました。

（2）FreeNXが使えるまで

パッケージはインストールしているので管理者権限で

cd /usr/local/NX

./bin/nxsetup —install

これで準備できました。あとはクライアント側に「/usr/local/NX/home/.ssh/client.id_dsa.key」をコピーします。

次にクライアントの準備です。クライアントにはNOMACHINE(http://www.nomachine.com/)を使います。

参考：http://www.atmarkit.co.jp/flinux/rensai/linuxtips/926usefreenx.html

Try WiMAX ３回目

Nov 21st, 2009

風邪がまだ治らない林です。

一度病院にいったほうがいいのでしょうか。

＝＝＝＝＝＝＝

「Try WiMAX」とは、UQコミュニケーションズが行っている次世代無線通信規格WiMAXを使った無線通信インフラの販促キャンペーンの一つです。

月額５０００円程度のサービスを１５日間無料で体験できるものです。

http://www.uqwimax.jp/service/trywimax/

東京に住んでいるので６月のサービス開始から２回申し込んで使っています。

１回目はダメで、２回目はそこそこで今回が３回目です。

私の住んでいる地域は１２月から正式にサービスインするらしいのですが、現状で下り４M、上り１Mと２回目と変わらずそこそこな通信速度が出ています。

http://wimax.noshikumi.com/

平均的にみてそこそこなんですが、通信品質（一定の通信速度がで出ていれば良い）はノコギリの刃みたいになっています。

これがサービスイン前だからなのかこれからもそうなのかは不明ですが。

１２月の正式なサービスインになったら4回目にチャレンジしてみようと考えています。

＝＝＝＝＝＝＝＝

こぼれ話ですが、Try WiMAXは９０日間は再度契約できないらしいです。

契約書にはそんなこと書いてないし、販促員もそんなことは一言もいってませんでしたｗ

このことが分かったのは、３回目のとき。

池袋のビックカメラで申し込んで受け渡しのときに店員が「お客様は以前に２回申し込まれてますよね？困るんですよ。こういうことされたら」みたいなことを言ってきました。

そんなつもりは全くなかったし、初めて聞いたことで貸せないなら良いよと思いながら、とりあえず「そうですか。何も販促員は何もいってませんでしたよ？」と言ってみた。

何回か問答を繰り返した末に「今回だけは特別ですから。次回からは難しくなるかもしれませんよ」と言われ、丁寧に梱包してある機器を押し付けられるようして持って帰ってきましたｗ

１０分くらい受け渡しカウンターで問答を（周囲がうるさいので）大声で繰り返していた中で店員が「あそこの店では特別に貸してくれた」と言われては困るんですけどみないなことを言っていたが、使わせてもらうことにしました。

あと正式にサービスインすると前回から９０日未満でも貸してくれる可能性があると言っていたので、4回目も行けそうですｗｗ

NicoCache + SHOUTcast

Nov 9th, 2009

風邪がまったく治らない林です。辛いです。

暇をみてはニコニコ動画をみながらプログラミングしているわけですが、こっち（東京）にくる前に勢いで買ったソニー「mylo」の出番がめっきりないのでなんか活用できる方法がないか考えてみした。

（今となってはmyloはiPod touchやiPhoneの出現で存在意義が無くなった感があるのは秘密ですw)

myloには「mylo widget」というものがあっていつくかプリインストールされています。

その中に「SHOUTcast widget」というのがあり、面白いものがあるなーとは思っていたのですが、聴きたい音楽が流れている番組？をサイトから見つけるのに一苦労するのでほとんど使っていませんでした。

最近、NicoCacheを管理するためのWEBページを改良して削除された動画もブラウザで再生できるようにしたんですが、これはmyloではみれないんですよ。

しかもMacbookで再生するとFANが唸りを上げ始めるので、myloでなにかできないか、と考えていると

mylo –> SHOUTcast Widget –> SHOTcastは自家製インターネットラジオできる –> 自家製SHOUTCastを作る –> NicoCacheでキャッシュした動画（音声のみ）をSHOUTcastで流す –> myloで聞ける♪

という筋書きを立てました。

＝＝＝＝＝＝＝＝

ググりながら色々探してみると「SHOUTCast Server + Winamp + edcast DSP」の組み合わせが一番いいらしいので、

超図解！誰でも始められるネットラジオ SHOUTcast編 (http://www.atamanikita.com/start-shoutcast/)

に従ってインストールをしてとりあえず動作を確認。

myloで聞けましたよ♪

ここまでくると欲が出てきます。常に音楽を流し続け、聞きたいときに聞けるようにするという欲がw

＝＝＝＝＝＝＝＝

常に音楽を流し続けるにはPCの電源を入れっぱなしにしておかなければいけないということになるわけですが、現実的な話じゃないです。

そこで家にあるサーバ上で動かすことにしましたが、作業を始めるといくつか問題が出てきて、ググって調整しての繰り返しでしたが、なんとか動くようになたのでメモっておきますw

１．オーディオデバイスがない

今回はWindows Hyper-V Server 2008 R2の上で動くWindows XP Pro SP2上にインストールするんですが、もちろんサウンドデバイスがありませんので、winampで再生するとエラーが出ます（涙

そこで音楽オーサリングソフトに付属している「仮想オーディオデバイス」とかでいけるんじゃねーとググってみるとありました。入れてみるとwinamp動きました。

http://ch.nicovideo.jp/community/co106496

上から「Avnex Virtual Audio Device」を使いました。

２．SHOUTcastのステータスで日本語が文字化けする

SHOUTcast serverのインデックスページの文字コードがShift-JISになってないので

http://www11.atwiki.jp/chaosradio/pages/21.html#id_119a9295

を参考して編集しました。

でも、myloではまだ文字化けしているんですが（涙

３．再生するとCPU使用率が100%に張り付く

どこからか仮想化して動画再生なんてもってのほかとか聞こえそうでが、やるしかありません。

winampの設定でビデオを描画しないようにウインドウを閉じたまま再生するようにしましたが変化なし。

winampではフルインストールするとプラグインとしていくつかcodecが入っているんですが、こいつが問題なのかと思い拡張子がMP4,FLV（SWFはうまく再生されないので保留）をDirectShowに担当させてDirectShow Spliter & FilterとCodecを入れると50%以下に収まりました♪

入れたものは

・ffdshow ( http://sourceforge.jp/projects/ffdshow-tryout/ )

・Haali media splitter ( http://www.gigafree.net/media/codec/haalimediasplitter.html )

・CoreAVC Codec ( http://www.free-codecs.com/download/codec.htm )

・FLV Splitter 1.0.0.5 ( http://d.hatena.ne.jp/wocota/20090524/1243152696 )

半分くらいはよくわからずですが、mp4の方が使用率が低くなったのは良かったと思っていますｗ

winampは豊富なプラグインが用意されています。
