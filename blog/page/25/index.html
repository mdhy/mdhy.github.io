勝手に動き出す

Jun 8th, 2008

　数日前のことmacbook起動しなくなりました。 　そして本日のこと，リストアCDも手元に届き，USBハードディスクドライブにバックアップも大体完了していた。 　最終確認にバックアップ漏れがないか確認するために，電源ボタンを押した。 　案の定，シングルユーザモードに切り替えるボタンを押すのを忘れて，エンドレスループに入ると思いきや・・・ 　・・・ 　起動しちゃったよww 　＝＝＝＝＝＝ 　今回試みたバックップ作業の備忘録 　シングルユーザモードへの切換え 　電源ボタンを押して，command + s　を押し続けるとCLI(コマンド・ライン・インターフェイス)に移行します。 　ファイルシステムを修復するらしい 　# fsck -fy 　シングルユーザモードでもリードオンリーであるため，ルートディレクトリ下を書込み可能にする 　# mount -uw / 　USBハードディクスドライブをマウントするための作成 　# midir -p /mnt/usb 　FATでフォーマットされているため，msdosでマウントする。 　# mount -t msdos /dev/disk1s1 /mnt/usb 　ディレクトリ単位でまとめる（tarでまとめると，一部が壊れると全て取出せなくなるため） 　# find Backups -depth -print | cpio -o > Backups.cpio 　FATでは4GBを超えるファイルは書き込めないので分ける 　# split -b 4000000000 Backups.cpio /mnt/usb/Backups.cpio. 　分割したファイルを結合する 　# cat /mnt/usb/Backups.cpio.?? > Backups.cpio 　まとめたファイルを展開する 　# cpio -idm < Backups.cpio
