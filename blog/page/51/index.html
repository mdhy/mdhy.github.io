ヘリコプターの操縦免許は結構お金がかかるらしい・・・ 　親愛なる友人からアパッチを乗りこなせないというヘルプコメントをもらったので、乗り方を少し紹介しようと思います〜 　アパッチは皆さんご存知の通り、これではなく、こっちですねぇ 　そうです。世界で最も使われているであろうWEBサーバの一つです。 　ヘルプの内容としては、Basic認証とMySQLはPHPでしか使えないかということ・・・ 　まず、Basic認証から 　@IT-Apacheでユーザー認証を行うには（Basic認証編） 　まず、htpasswdを使ってパスワードファイルを作る。 　# htpasswd -c (パスワードファイルパス) (追加アカウント) 　追加は、 　# htpasswd (パスワードファイルパス) (追加アカウント) 　（パスワードファイルパス）については、相対パスでも絶対パスでも可。 　上で生成したパスワードファイルを使って、Basic認証を行うためには、Apacheの設定ファイルであるhttpd.confを編集してやる。 　httpd.confでBasic認証をしたい・・・や・・・内に以下のようなものを書いてやれば実現できる。 （ディレクトリの設定省略） AuthType Basic AuthName “Secret Zone” AuthUserFile /etc/httpd/.htpasswd Require user secret 　Requireの部分については、ここを見て、違いから理解してもらいたい。 　Basic認証に関しては以上。 　次は、MySQLはPHPからしか使えないことはない。 　PerlにもCPANにMySQLと通信するためのモジュールがある。 　DBD::mysql 日本語説明はこちら 　使い方の例 　今回はこれくらいで、DBDについては、DBDやMYsqlを一緒にググってやれば、わんさかでくるｗ 　まぁ頑張って、理解して下さい〜 　何かあったら、こめんとください。
