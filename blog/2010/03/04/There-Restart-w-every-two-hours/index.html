あれ！？２時間おきに再起動ｗ

Mar 4th, 2010

きのう、地デジ観るPCが起動時にFANの回転数が高くなるとうるさいくなるので交換するために仕事帰りに代替パーツを買って帰ったんですが違うパーツが壊れたいたことが分かったて、買ったパーツもグラボが邪魔して取り付けできないことがわかって、あれ？！ってなった林です。

いまアパート（台所）にMicrosoftの「Hyper-V Server 2008 R2 RC版」をインストールして使っているPC（以下、太朗くん）が1台あるんですが、地デジ観るPCにどうやって代替パーツを付けようか悩んでいると太朗くんが再起動しているようです。

あれ！？なんでｗ

自動更新は無効にしてあるはずなのに、さいきん使ってないから拗ねたか？なんて妄想しながらイベントログを見てみると3月に入ってから2時間おきに再起動しているｗｗ

そういえばWindows 7 RCは3月1日から2時間おきにシャットダウンするようなメールがきていたけど、Hyper-V Server 2008 R2 RCは観てないぞ、メール読んでないから知らないけど？なんて考えながら20時に再起動する前に仮想マシンをエクスポートして正式版に入れ替えました♪

20時からインストール初めて23時には仮想マシン4台をインポートして動作確認できました。

やっぱりMicrosoftの製品は使いやすいですねーお高いですけど；；

ちなみにHyper-V Server 2008 R2 は無償で提供されています。

ダウンロードの詳細 : Microsoft® Hyper-V™ Server 2008 R2 : http://bit.ly/3Gu3vL
