Jan 24th, 2008

　コレッて動くの？ 　報告者の第一稿を先生に見せて、自分査読をしたわけですが・・・ 　一夜明けて、脳細胞と目の筋肉を休ませることのできた今は、参考文献の整理と図が原因で、できる余分な「空白」の始末をしながら、記事を書いているわけです。 　雪がワンサカ降っているのにサマータイヤで学校に来たことを半分後悔しながら、朝風呂に入った時にふと考えた決め台詞を思い出しました。 　「きみの行動は、何も書かれていない関数と同じで、無意味だ！」 　ある意味潔癖なニオイがしますが、頭から離れませんｗ 　「なにも書かれていない関数と同じで」っていうのは 　void dummy(void){ 　} 　とか 　function dummy(){ 　} 　とか 　public void dummy(void) 　{ 　} 　なのか？？と自問して、頭から離れませんｗｗ 　void dummy(void){ 　　;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;;; 　} 　もう、こうなると、邪魔なだけですよｗｗｗ 　あーでも、これってコンパイルできたっけ？
