Vmware Server 2.0 + Windows 7

Nov 15th, 2008

久しぶりに洗濯物をおひさまで乾かそうと思って干してみたら急に曇ってきたのでビビッて取り込んだらまた晴れるといった自然の気まぐれに振り回された林です。

＝＝＝＝＝＝

土日は家に引き込んでいますが、何をしているかというとボーとしながら現場でマイクソフト社のWINDOWS製品を多く扱っているのでその勉強をしてます。

手元によくできたWindows 7(マイクロソフト社の次期発売のオペレーティングシステム)のPre-BETAがあるので、インストールしてみした。

巷には仮想化という技術が浸透してきていて、今では素晴らしい技術の数々が無料で使えるまでになっています。

今回はノウハウもあってWindows Vistaを動かすことのできるVMware社のVMwarePlayer2.0を使ってみることにしました。

何をやっても一筋縄ではいかない私なのでインストールはできましたが、なぜかネットワークアダプタが認識されてない；；

ググってみるけど、もちろんのことながら参考になる情報は全くないという状況でした＞＜

それでもわかったことはWMwareToolsというものがWMwareServerから手に入れることができて、たぶんネットワークアダプタのドライバも入っているだろうということで、WMwareServer2.0をダウンロードしてインストールしてみることにしました。

WMware Server 2.0 (http://www.vmware.com/jp/products/server/)

WMware Serverは初期の1.x系から使っていきましたが、2.0系からガラリと仕様が変わって今流行？のWEBアプリケーションでコントロールするみたいです。

ちょっと見た感じでは、Tomcat3で動くJAVAアプリのようのです。

（スクリーンショットを貼りまくって素晴らしさをアピールしたいところですが、このブログは未だに画像が貼れません。）

気になるかたは是非、使ってみてください↑

＝＝＝＝＝＝

そんなこんなでWindows 7をVMware Server上にインストールしていきます。

ある記事（http://japan.cnet.com/special/story/0,2000056049,20383252,00.htm%EF%BC%89%E3%81%AB%E3%81%AF%E3%82%A4%E3%83%B3%E3%82%B9%E3%83%88%E3%83%BC%E3%83%AB%E3%81%8C%E3%81%8B%E3%81%AA%E3%82%8A%E9%80%9F%E3%81%84%E3%80%81%E8%B5%B7%E5%8B%95%E3%82%84%E7%B5%82%E4%BA%86%E3%81%8C%E9%80%9F%E3%81%84%E3%81%A8%E3%81%84%E3%81%86%E3%81%93%E3%81%A8%E3%81%A7%E3%81%97%E3%81%9F%E3%80%82

そのとおりでした。速いです。

＝＝＝＝＝

もちろんこの記事もWindows 7のIE8で書いています♪(途中からFirefox3.0.0.4で書いていますが、IE8よりFirefoxの方がスクロールからして快適です。)

今はリモートデスクトップで使っていますが、コンソールからも快適です。

それに項目の表示は英語なんですが、標準IMEで日本語入力・表示が可能なんですよ。（まぁ当たり前ですがw）

＝＝＝＝＝＝＝

まだ何も入っていないので軽いのでしょうが、Firexox使って500MB以上メモリを消費しています。

まぁWindows XP Pro上のWMware Server 2.0で十分動くなら文句はない感じですが。

Window7出たら買ってもいいですね♪

ちなみに、Kingsoft Internet Security Uはインストールはできて、リアルタイムウイルスガードは動いているようです。Firefoxも使えています。

なんか３０日間は使えるようなので使っていきます。

ちなみにGUIはWindowsVistaそのものでした。

なにか見つけたらまた書きます。
