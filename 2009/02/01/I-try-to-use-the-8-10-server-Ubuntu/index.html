Ubuntu 8.10 Serverを使ってみる

Feb 1st, 2009

叔父さんの家からFreeNXを使って自宅のサーバを操ってこの記事を書いている林です。

サーバを新調してからCPUのコアが４つになったので何かOSをインストールしてやろうと思い、debian（デビアン）を選択しようと思いましたが以前使ったことのあるUbuntu（ウブントゥ）がdebianとすごく良く似ているということで

Ubuntu 8.10 Server

を手に入れて、Xenの上にインストールしました。

UbuntuのServer版はインストールの時に指定できるサーバサービスを選択してインストールしないと何も入っていなんですよw

所感として

・空いているポートは６８のみ。

・コンパイラとか一切無し。

・プロセスで８０行で表示できる範囲に収まる。

・rootアカウントが存在しない。（sudoコマンドを使う）

・ネットに参考資料が豊富にあるので困ることはない。

・openvpnサーバを移設してみたけど、問題が起きなかった。

といった感じです。

＝＝＝＝＝

FreeBSDぽい匂いがして、とても好みです。

Desktop版もあるし、LiveCD版もあるのでお試しあれ。

Ubuntu Japanese Team : http://www.ubuntulinux.jp/
